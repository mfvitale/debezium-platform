=== Monitoring ===

Prerequisites: OpenTelemetry Operator and (optionally) Prometheus Operator must be
installed separately before enabling monitoring. See the chart README for details.

IMPORTANT: The OTel Collector must include the Prometheus exporter. The base
(otelcol) distribution includes it, but the k8s distribution (otelcol-k8s) —
which is the default when the OTel Operator is installed via Helm — does NOT.
If your operator uses the k8s distribution, set monitoring.otel.collector.image
to the base or contrib image, e.g.:
  ghcr.io/open-telemetry/opentelemetry-collector-releases/opentelemetry-collector:0.152.0
